## 二、核心功能模块

### 模块一：日度监控数据日报（每日10:00-10:30）

#### 触发条件
用户告知"数据已更新"或"开始生成日报"

#### 执行步骤

**Step 1: 读取数据**
- 读取路径：`D:\HY-日度数据监测\HY-日志导出月度汇总\视频审核操作日志-YYYY.MM.DD.xlsx`
- 筛选审核人包含"易盾BPO"的记录

**Step 2: 计算整体业务指标**

| 指标 | 计算公式 |
|------|---------|
| 实际审核总量 | 总记录数 |
| 转化后审核总量 | Σ(视频时长>1h ? 实际量×1.5 : 实际量×0.78) |
| 通过量 | 审核结果="通过"的记录数 |
| 通过（风险）量 | 审核结果="通过（风险）"的记录数 |
| 不通过量 | 审核结果="不通过"的记录数 |
| 违规率 | (不通过量/实际审核总量) × 100% |
| 平均排队时长 | 平均(进入审核时间 - 工单生成时间) |
| 平均审核倍率 | 平均((审核完成时间 - 进入审核时间) ÷ 视频时长) |
| 当班次人数 | 去重后的审核人数 |
| 人均转化后审核量 | 转化后审核总量 / 当班次人数 |

**Step 3: 计算当班次人员指标**

| 指标 | 计算公式 |
|------|---------|
| 个人审核量 | 每人审核记录数 |
| 个人转化后审核量 | Σ(视频时长>1h ? 1.5 : 0.78) |
| 个人通过量 | 每人审核结果="通过"的记录数 |
| 个人不通过量 | 每人审核结果="不通过"的记录数 |
| 个人违规率 | (个人不通过量/个人审核量) × 100% |
| 个人平均排队时长 | 平均(进入审核时间 - 工单生成时间) |
| 个人平均审核倍率 | 平均((审核完成时间 - 进入审核时间) ÷ 视频时长) |

**Step 4: 异常预警判断**

| 预警类型 | 判断条件 | 目标值 |
|---------|---------|--------|
| 产能异常 | 人均转化后审核量 < 230 | 230条/人/天 |
| 违规率异常 | 整体违规率 > 15% 或 < 5% | 15%左右 |
| 排队时长异常 | 平均排队时长 > 2分钟 | 2分钟以内 |
| 审核倍率异常 | 平均审核倍率 > 1.5 或 < 0.5 | 0.5-1.5 |
| 通道异常 | 非4个正确通道的数据 | 4个正确通道 |

**正确通道（4个）**：
1. 虎牙视频
2. 虎牙直播回放切片视频
3. 虎牙AI礼物特效视频
4. 虎牙一起看视频

**Step 5: 月度账单数据落库**
- 提取字段：vid、所属应用、视频时长、上传时间、工单生成时间、进入审核时间、审核完成时间、审核结果、审核人
- 添加"数据日期"列标识
- 写入路径：`D:\虎牙数据汇总\虎牙结算数据\26年结算\6月结算\6月人审量.xlsx`
- 追加到现有Sheet中

**Step 6: 生成日报并发送到企业微信群**

日报格式：
```
📊 【虎牙视频审核日报】YYYY-MM-DD

📈 整体业务情况
- 实际审核总量：XXX 条
- 转化后审核总量：XXX 条
- 通过量：XXX 条
- 通过（风险）量：XXX 条
- 不通过量：XXX 条
- 违规率：XX.X%
- 平均排队时长：XX 分钟
- 平均审核倍率：XX.X
- 当班次人数：XX 人
- 人均转化后审核量：XX 条/人

👥 当班次人员情况（所有人员）
| 姓名 | 审核量 | 转化后审核量 | 通过量 | 不通过量 | 违规率 | 平均排队时长 | 平均审核倍率 |
|------|--------|------------|--------|---------|--------|------------|------------|
| XXX | XXX | XXX | XXX | XXX | XX% | XX分钟 | X.XX |
```

## 五、注意事项

1. **数据时效性**：日报数据必须在前一日审核完成后导出，确保数据完整性
2. **人员筛选**：所有计算必须基于审核人包含"易盾BPO"的记录
3. **转化规则**：视频时长>1h按1:1.5转化，<1h按1:0.78转化
4. **审核倍率**：审核倍率 = (审核完成时间 - 进入审核时间) ÷ 视频时长
5. **违规率**：违规率 = 不通过量 / 实际审核总量
6. **通道异常**：只认可4个正确通道，其他通道需预警
7. **月度账单落库**：日报发送后自动执行，确保数据及时归档
8. **看板更新**：月度账单落库后自动执行，确保数据实时性
9. **月度结算**：每月1号优先执行月度结算，再生成当日日报
