## 🔢 计算规则

### 筛选条件
| 规则 | 说明 |
|------|------|
| 审核人 | 只统计审核人包含 **"易盾BPO"** 的记录 |
| 数据范围 | 当日所有完成审核的数据 |

### 指标公式
| 指标 | 公式 |
|------|------|
| **实际审核总量** | 总记录数 |
| **转化后审核总量** | Σ(视频时长>1h ? 1.5 : 0.78)，**向下取整** |
| **通过量** | 审核结果 = "通过" |
| **通过（风险）量** | 审核结果 = "通过（风险）" |
| **不通过量** | 审核结果 = "不通过" + "已删除" |
| **违规率** | 不通过量 / 实际审核总量 × 100% |
| **平均排队时长** | 平均(审核完成时间 - 进入审核时间) / 60（分钟），单位：分 |
| **平均审核倍率** | 平均( (审核完成时间 - 进入审核时间)(秒) / 视频时长(秒) )，只取审核耗时≤3600秒的有效数据 |
| **当班次人数** | 去重后的审核人数 |
| **人均转化后审核量** | Floor(转化后审核总量 / 当班次人数) |

### 人员指标（每人）
| 指标 | 公式 |
|------|------|
| **个人审核量** | 每人记录数 |
| **个人转化后** | Σ(视频时长>1h ? 1.5 : 0.78)，向下取整 |
| **个人通过量** | 审核结果 = "通过" |
| **个人不通过量** | 审核结果 = "不通过" |
| **个人违规率** | 个人不通过量 / 个人审核量 × 100% |
| **个人排队时长** | 平均(审核完成时间 - 进入审核时间)(分)，≤60分钟有效 |
| **个人审核倍率** | 平均(审核耗时(秒) / 视频时长(秒))，≤3600秒有效 |

### 预警标准
| 预警类型 | 正常 | 警告 |
|----------|:----:|:----:|
| 人均转化后审核量 | ≥ 230 🟢 | < 230 🔴 |
| 个人违规率 | ≤ 15% 🟢 | > 15% 🟡 |
| 平均排队时长 | ≤ 2分钟 🟢 | > 2分钟 🔴 |
